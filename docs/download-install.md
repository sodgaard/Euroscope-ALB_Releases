# Download & Install



1. Download the latest release from the GitHub Releases page [Releases](https://github.com/sodgaard/Euroscope-ALB_Releases){target="_blank" rel="noopener"}

2. Unzip to your EuroScope plugins folder. Make sure you have the **alb.dllÿ** and the **alb-config.json** in the folder.

3. Start EuroScope and enable the ALB plugin.



Having issues? See [Troubleshooting](troubleshooting.md).
