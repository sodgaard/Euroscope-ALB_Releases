# Download & Install

1. Download the latest release from [Releases](https://github.com/sodgaard/Euroscope-ALB_Releases/releases){target="_blank" rel="noopener"}.

2. Unzip to your EuroScope plugins folder.

3. Verify that the plugin folder contains at least:

   - `ALB.dll`
   - `ALBCore.dll`
   - `alb-config.json`

4. The following files may also be present and are normal:

   - `ALBLoader.ini`
   - `alb-config.default.json`

5. In EuroScope, load or enable `ALB.dll` as the plugin entry point.

6. Start EuroScope. If the ALB window does not open automatically, use:

   ```
   .alb open
   ```

Having issues? See [Troubleshooting](other/troubleshooting.md).
